# RT3DPRINT Flask Project

This is a Flask-based 3D Printing Request Management System (RT3DPRINT).

## Quick start

1. Copy `.env.example` to `.env` and set values (MySQL and Gmail app password).
2. Install requirements: `pip install -r requirements.txt`
3. Create MySQL database and tables (see SQL schema in project description).
4. Run: `python app.py`
5. Default admin username: `admin`, password from environment `ADMIN_PASS` or in .env (default: rThanush@0828)
